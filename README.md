# Бот «Признавашки НФ УУНИТ»

Анонимный бот для Telegram-канала. Принимает анонимки от пользователей, отправляет на модерацию, публикует в канал.

## Быстрый старт

### 1. Установить зависимости

```bash
npm install
```

### 2. Настроить `.env`

```env
BOT_TOKEN=     # токен бота от @BotFather
MOD_CHAT_ID=   # числовой ID чата модераторов (см. ниже)
CHANNEL_ID=    # username канала: @priznavashkiyynit
```

### 3. Запустить

```bash
npm start
```

---

## Как получить MOD_CHAT_ID

1. Добавьте бота в группу модераторов с правами администратора.
2. Напишите любое сообщение в эту группу.
3. Откройте в браузере:
   ```
   https://api.telegram.org/bot<BOT_TOKEN>/getUpdates
   ```
4. В ответе найдите `"chat": {"id": -100xxxxxxxxxx}` — это и есть `MOD_CHAT_ID` (отрицательное число).

## Как получить CHANNEL_ID

Для публичного канала используйте `@username` (например, `@priznavashkiyynit`).

Для приватного канала: перешлите любое сообщение из канала боту `@userinfobot` — он покажет числовой ID.

## Команды модераторов

| Команда | Описание |
|---------|----------|
| `/stats` | Статистика (подано / опубликовано / отклонено / отложено) |
| `/unban <user_id>` | Разбанить пользователя |
| `/bans` | Список забаненных пользователей |

---

## Требования

- Node.js 18+
- Бот должен быть **администратором** канала с правом публикации сообщений
- Бот должен быть **участником** чата модераторов
